O que e Redis?

Redis é um acrônimo de Remote Dictionary Server (servidor de dicionário remoto). É focado no alto desempenho, contendo características marcantes, como a agilidade de acesso e armazenamento de informações (devido sua estrutura de funcionalidade).

Como funciona o Redis?

Ele trabalha com o armazenamento de valores chaves e banco de dados na memória. Dessa forma, a cada entrada de informações no banco se cria uma chave, e com essa chave conseguimos acesso aos dados(tornando mais simples o escalonamento da estrutura).

Vantagens

Devido ao seu baixo nível de código que demanda na hora de gerenciar dados de forma geral, facilita o uso e se torna prático. Sua manipulação de dados (de forma nativa), mantém a estrutura do código intacta deixando simples e rápido.

Desvantagens

Por se tratar de um banco de dados persistente em disco, tendo alocação direta na memória RAM, é primordial considerar sua utilização para evitar problemas.
